# 强化学习实现

[English](README.en.md)|[中文](README.md)

本仓库致力于实现著名书籍《强化学习：导论》（第二版，完整草案，2017年11月5日）的任务，该书由Richard S. Sutton和Andrew G. Barto撰写，由麻省理工学院出版社出版。

## 项目目标

本项目的主要目标是通过实际的编程练习使书中介绍的理论概念和算法生动起来。我们旨在通过开发和探索书中描述的各种模型和技术，增强对强化学习的理解。
